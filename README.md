# 🏆 Family Responsibility & Virtue System

This project is a **Full-Stack** mobile application that digitizes traditional family task management using gamification principles.

---

## 🚀 About the Project
The Family and Profile Management System aims to optimize responsibility sharing among family members and help children develop consistent habits.

---

## ✨ Key Features

* **📸 Photo-Proof System:** Tasks are verified with photo evidence and submitted for parental approval.
* **🎖️ Virtue Badges:** Specialized badges are earned for abstract values such as honesty and kindness.
* **⚖️ Weekly Decision Maker:** The first child to earn a "Virtue Badge" each week gains the authority to determine the details of the family weekend activity.
* **📈 Performance Analytics:** Long-term development is tracked via an annual achievement grid.

---

## 🛠️ Tech Stack
The project is built on a modern and scalable three-tier architecture:

* **Frontend:** SwiftUI (iOS) - MVVM Architecture
* **Backend:** Python & Flask (REST API)
* **Database:** PostgreSQL (Relational Model)
* **Animations:** Lottie (JSON-based vector animations)

---

## 🗄️ Database Design
To ensure data integrity, the system utilizes 10 relational tables (Families, Roles, Users, Tasks, Badges, etc.).

# 🏆 Aile İçi Sorumluluk & Erdem Sistemi

Bu proje, geleneksel aile içi görev yönetimini oyunlaştırma prensipleriyle dijital bir platforma taşıyan **Full-Stack** bir mobil uygulamadır. 

---

## 🚀 Proje Hakkında
Aile ve Profil Yönetim Sistemi; aile bireyleri arasındaki sorumluluk paylaşımını optimize etmeyi ve çocuklara düzenli alışkanlıklar kazandırmayı amaçlar. 

### ✨ Öne Çıkan Özellikler
* **📸 Fotoğraflı Kanıt Sistemi:** Görevler fotoğraf kanıtı ile doğrulanır ve ebeveyn onayına sunulur. 
* **🎖️ Erdem Rozeti:** Dürüstlük ve yardımseverlik gibi soyut değerler için özel rozetler kazanılır. 
* **⚖️ Haftalık Karar Verici:** Haftanın ilk erdem rozetini alan çocuk, hafta sonu etkinliğini belirleme yetkisi kazanır. 
* **📈 Performans Analizi:** Yıllık başarı takvimi üzerinden uzun vadeli gelişim takibi yapılır.

## 🛠️ Teknik Yığın (Tech Stack)

* **Frontend:** SwiftUI (iOS) - MVVM Mimari
